Plan out the next milestone in detail using a planning agent.

1. Read VISION.md and MILESTONES.md.
2. Identify the next milestone that is `planned` or `in-progress` but
   lacks a detailed task breakdown.
3. Confirm with the user which milestone to plan before proceeding.
4. Use the Plan subagent to:
   - Break the milestone into concrete, independently-completable tasks.
   - Write acceptance criteria for each task.
   - Flag any dependencies between tasks.
   - Identify risks or open questions that should be resolved first.
5. Present the plan to the user for review. Revise based on feedback.
6. Once approved, write the task breakdown into MILESTONES.md under the
   relevant milestone.
7. Update STATUS.md "Next Up" with the first 2-3 tasks.
